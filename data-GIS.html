Accessing Spatial data from R

  • Web Map Services (WMS): data in raster format e.g. images (e.g. .tif)
  • Web Map Tile Service(WMTS) : same as WMS raster but images are precalculated as tile
  • Web Feature Service (WFS) : data in vector format (e.g. .shp).
